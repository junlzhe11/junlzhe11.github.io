# Apple Calendar

Category: Habit
Tags1: Knowledge sharing
Tags2: Habits

导入苹果日历（Apple Calendar）可以通过多种方式实现，具体取决于你要导入的是

- 需要订阅的动态日历链接
- 同步的其他平台账户（如 Google 或 Outlook）。
- 单个日历文件（.ics）

以下是常见的几种导入方法和共享日历的方法

# 导入方法

### 方法一：订阅日历链接（Webcal / iCal 链接）

如果你有一个以 `http://`、`https://` 或 `webcal://` 开头的日历订阅链接（通常用于节假日、体育赛事排期或公司排班），导入后日历会自动更新。

- **🍎 在 Mac 上：**
    1. 打开 **“日历”** 应用。
    2. 点击顶部菜单栏的 **文件 > 新建日历订阅**。
    3. 粘贴你的日历链接，点击 **订阅**。
    4. 在弹出的设置菜单中，你可以自定义名称、颜色，以及**自动刷新的频率**（建议设置为“每天”或“每小时”），最后点击 **“好”**。
- **📱 在 iPhone / iPad 上：**
    1. 打开手机的 **设置 > 日历 > 帐户**。
    2. 点击 **添加帐户 > 其他**。
    3. 在“日历”版块下，点击 **添加已订阅的日历**。
    4. 在“服务器”栏中粘贴你的日历链接，点击 **下一步** 并保存。

---

### 🍎 通过 iCloud 网页端管理与分享指南（免客户端）

如果你在 Windows 电脑上工作，且不想安装任何 Apple 客户端软件，直接使用浏览器访问 iCloud 官网是最纯净的操作方式。

### 1. 登录与进入

1. 在浏览器中访问 [iCloud.com](https://www.icloud.com/)。
2. 使用你的 Apple ID 登录。
3. 在应用网格中点击 **“日历”** 图标。

### 2. 公开自己的日历（生成订阅链接供他人使用）

如果你想把自己的日历（如“秋招行程”或“课程表”）分享给别人：

1. 在左侧侧边栏中，将鼠标悬停在目标日历名称上。
2. 点击右侧出现的 **共享图标**（类似雷达波的小图标）。
3. 在弹出的窗口中勾选 **“公开日历” (Public Calendar)**。
4. **获取链接**：系统会生成一串以 `webcal://` 开头的链接。点击“复制链接”即可发送给他人。
    - *注：他人拿到此链接后，可以按照上方“Windows 订阅”的方法将其导入。*

### 3. 网页端快速同步技巧

- **即时生效**：你在网页端修改的任何事件（双击日期即可新建），会通过云端瞬间同步到你的 iPhone、iPad 和 Mac 上。
- **查看共享状态**：如果某个日历名旁边有一个小人头图标，说明该日历目前正处于“公开分享”状态。
- **隐私控制**：如果你想停止分享，只需再次点击共享图标，取消勾选“公开日历”，之前的链接就会立即失效。

### 4. 局限性提示

- **添加他人订阅**：目前 iCloud 网页版主要用于管理“自己的”日历和“已有的”订阅。若要添加一个新的 `webcal://` 外部订阅链接，建议优先在 iPhone 或 Mac 上操作一次，网页端随后会自动同步显示。

### 方法二：同步其他账户的日历（Google / Outlook / Exchange）

如果你在其他平台有现成的日程，不需要手动导出，直接绑定账户即可实现双向同步。

- **🍎 在 Mac 上：**
    1. 打开 **系统设置 > 互联网帐户**（旧版本 macOS 为“系统偏好设置”）。
    2. 点击 **添加帐户**，选择你要同步的服务（如 Google、Microsoft Exchange 等）。
    3. 完成账号登录授权后，确保勾选了 **日历** 选项。
- **📱 在 iPhone / iPad 上：**
    1. 打开手机的 **设置 > 日历 > 帐户**。
    2. 点击 **添加帐户**，选择服务商（Google、[Outlook.com](http://outlook.com/) 等）并登录。
    3. 登录成功后，将 **日历** 的开关保持开启状态，然后点击 **保存**。
- **💻 在 Windows 上（自带日历 / Outlook）：**
    1. 打开 Windows 系统的 **“日历”** 应用（或新版 Outlook）。
    2. 点击左下角的 **⚙️（设置）** 图标，选择 **“管理帐户”** (Manage Accounts)。
    3. 点击 **“添加帐户”** (Add account)，选择你要同步的服务商（如 Google、iCloud 等）并登录。
    4. 完成授权后，该账户的日程即可无缝双向同步到你的 Windows 电脑上。

### 方法三：导入 .ics 日历文件

如果你下载了 .ics 格式的日历文件（通常用于课程表、航班信息或特定活动），可以通过以下方式导入：

- **在 Mac 上：**
    1. 最简单的方法是直接双击该 .ics 文件。系统会自动打开“日历”应用。
    2. 在弹出的窗口中，选择你要将这些日程添加到的目标日历（如“家庭”或“工作”），然后点击“好”。
    3. **或者：** 打开“日历”应用，点击顶部菜单栏的 **文件 > 导入**，选择你的 .ics 文件并确认。
- **在 iPhone / iPad 上：**
    1. 如果 .ics 文件在“文件”App、邮件或微信/备忘录中，直接点击打开该文件。
    2. 系统会展示日程列表。点击屏幕右上角的 **添加全部**（或单个添加）。
    3. 选择要存入的日历分类，点击 **完成**。
- **在网页端（Windows 电脑适用）：**
    1. 浏览器访问 [iCloud.com](http://icloud.com/) 并登录你的 Apple ID。
    2. 打开“日历”应用。
    3. 点击左下角（或右上角）的 **设置（齿轮图标） > 偏好设置**。
    4. 找到 **导入/导出** 选项，点击导入并选择本地的 .ics 文件。

# 共享方法

### 1. 方案一：iCloud 分享（Apple 用户间的“亲儿子”待遇）

如果你和对方都是 Apple 全家桶用户，这是最丝滑的选择。

**优点：**

- **深度集成**：对方点击邀请后，日历直接出现在自带 App 里，甚至能收到 Siri 的智能提醒。
- **权限精细**：你可以设置对方是“只能看”还是“可以改”。如果你们一起做项目，选“可以改”，对方加个 Deadline 你这边也能看到。
- **实时性**：iCloud 的推播在 Apple 设备间几乎是秒级的。

**操作：**

1. 在 Mac 日历侧边栏，点击日历名右边的“小人头”。
2. 输入对方的 Apple ID（邮箱）。
3. 对方会在 iPhone 上收到一个弹窗邀请。

---

### 2. 方案二：Google (Gmail) 分享（万能适配器）

如果你不确定对方以后会不会换安卓机，或者你们需要在 Windows 网页端频繁查看。

**优点：**

- **兼容性无敌**：即使对方明天换了三星或是在实验室的 Ubuntu 电脑上，都能看。
- **Notion 友好**：如果你想把这个共享日历以后嵌入到 Notion 页面里，Google 的连接性更好。

**缺点：**

- **刷新稍慢**：在 iPhone 原生日历里同步 Google 的共享日历，有时会有几分钟的延迟。

---

### 💡 场景建议：你怎么选？

| 场景 | 推荐方式 | 理由 |
| --- | --- | --- |
| **项目搭档（都是 Apple 用户）** | **iCloud 私人分享** | 方便两人共同编辑进度，权限管理最安全。 |
| **班级/社团（大范围分享）** | **Google 公开链接** | 不管别人用什么手机都能订阅，不用一个个加好友。 |
| **给自己看（跨设备同步）** | **Google 账号同步** | 既然你已经在 Mac 登录了 Google，就让它作为主库。 |
